## ディレクトリ構成

この class4 ディレクトリ配下に配布された data ディレクトリを展開してください。

```
class4
├── data
│   ├── graph_small.png
│   ├── links_small.txt
│   ├── links.txt
│   ├── pages_small.txt
│   └── pages.txt
├── README.md
└── wikipedia_bfs.cc
```
c++実行環境が整っている方は
```class4/ にて実行
g++ wikipedia_bfs.cc -o wikipedia_bfs.out
./wikipedia_bfs.out
```
